# DOM Manipulation Example

This project is a simple example of **DOM manipulation using JavaScript**.
It allows users to enter their name and interact with colored boxes.

## Features

* A header that initially displays **"Hello"**
* An input field where the user can enter their name
* A **Greet button** that updates the header to **"Hello, [Name]"**
* Four clickable boxes:

  * Red
  * Blue
  * Green
  * Yellow
* Clicking a box changes its **background color** to the corresponding color

## Technologies Used

* **HTML** – Page structure
* **CSS** – Styling and layout
* **JavaScript** – DOM manipulation and event handling

## Project Structure

```
project-folder
│
├── index.html

```

## How to Run

1. Clone or download the repository.
2. Open the `index.html` file in any web browser.
3. Enter your name in the input field.
4. Click **Greet** to update the greeting message.
5. Click any color box to change its background color.

## Purpose

This project was created to practice:

* Basic **DOM selection**
* **Event handling**
* **Updating HTML elements dynamically with JavaScript**
